## The v8 ODE system — what's different from v4-v7

### Canonical ODE (from script header)

```text
α'' + α'/r - α/r² + ω²α = β(r)              [A-field]
β'' + β'/r - β/r² + ω²β = α(r) - λβ - 4gβ³   [J-field]
```

### Key technical specifications

| Spec | v8 value | v4-v7 value | Significance |
| --- | --- | --- | --- |
| Number of radial functions | 2 (α, β) | 4 (V, A, Q, J) | Eliminates excited-mode artifacts |
| Laplacian form | Vector: f'' + f'/r − f/r² | Scalar spherical: f'' + 2f'/r | Cylindrical/toroidal geometry, not spherical |
| Boundary conditions | **Slope BCs**: α(r_inner)=A0·r_inner, α'(r_inner)=A0 | Value BCs at r=0 | Field ~ A0·r near origin; analytic |
| Integration domain | [r_inner=0.02, R_MAX=12.0] | [0, R_MAX] with regularization | Avoids 1/r² singularity by offset |
| Method | `solve_ivp` (RK45 forward integration) | `solve_bvp` (collocation) | Forward shoot, not BVP |
| Lagrange multiplier λ_LM | None — λ is a fixed parameter (λ=1.0 for electron) | Free parameter | Removes one source of mode-selector instability |

### Neutral chaoiton reduction

When A_0 = 0 and ω = 0, the α-equation decouples (α=0 exact solution). The
β-equation becomes:

```text
β'' + β'/r - β/r² + λβ + 4gβ³ = 0
```

with β(0)=0, β'(0)=B0. Mass prediction: m ≈ λ × m_e.

## Step 3 — Three-system mapping (paper-math)

### v1 ODE (variant A4 + H1 + Q2, the 1.6918 result)

```text
α'' + (2/r)α' + ω²α = -γ/4
γ'' + (2/r)γ' + ω²γ =  4gγ³ - α/4
```

Volume element: 4π r² dr (full **spherical** integration)

Charge: Q = 4π ∫ ω(α² + γ²) r² dr
Hamiltonian: H = 4π ∫ [α'² + ω²α² + γ'² + ω²γ² + gγ⁴ − αγ] r² dr

### Sonnet's v8 ODE (canonical script)

```text
α'' + α'/r - α/r² + ω²α =  β
β'' + β'/r - β/r² + ω²β =  α - λβ - 4gβ³
```

Volume element: r dr (2D **cylindrical** integration)

Charge: Q_J = ∫ β² r dr
Hamiltonian: H = ∫ [α'² + β'² + α²/r² + β²/r² + 4gβ⁴] r dr

### v9 paper §5.1 (gradient ansatz)

```text
A = r̂ × ∇φ(r) cos(ωt)
J = r̂ × ∇ψ(r) sin(ωt)
```

Paper does not write explicit ODEs for φ, ψ. Geometry is implicit
(toroidal/cylindrical implied by r̂ × ∇ structure).

### Side-by-side comparison

| Feature | v1 A4+H1+Q2 | Sonnet v8 | v9 §5.1 |
| --- | --- | --- | --- |
| Coordinate system | Spherical (3D) | Cylindrical (2D, axisymmetric torus) | Cylindrical/toroidal (implied) |
| Radial Laplacian | f'' + 2f'/r (scalar spherical) | f'' + f'/r − f/r² (azimuthal vector cyl) | Not written |
| Coupling form | −γ/4 in α-eq, −α/4 in γ-eq | +β in α-eq, +α in β-eq | Not written |
| Mass-like term in γ/β eq | 4gγ³ | 4gβ³ + λβ | Not written |
| Volume element | 4π r² dr | r dr | Not written |
| Quartic in H | gγ⁴ | 4gβ⁴ | Not written |
| Coupling sign in H | −αγ | (none — direct coupling absorbed in EoM) | Not written |
| Best H/Q | 1.6918 (search across 6×4×4=96 variants) | 1.6890 (single derivation) | 1.6918 (claimed, no derivation shown) |
| Best gap | 0.30% | 0.090% | 0.30% (claimed) |

### Verdict

The three systems are **genuinely different physics**, not equivalent reductions
of one another:

1. **v1 (spherical) ≠ Sonnet (cylindrical)** — the volume elements (r² dr vs r dr)
   and Laplacian operators (scalar spherical vs vector cylindrical) make these
   distinct field theories. No change of variables maps one to the other while
   preserving solutions.

2. **Sonnet's geometry is cylindrical/toroidal.** This is consistent with:
   + The 1240 fm r_rms ≫ 386 fm Compton wavelength (extended object wrapping
     around a torus, not a 3D solid)
   + The L = ω Q_J angular-momentum form (natural for cylindrical symmetry)
   + The v9 §5.1 r̂ × ∇ ansatz structure (the cross-product with r̂ in cylindrical
     coords picks out the azimuthal component)

3. **v1's 1.6918 was a numerical coincidence,** not a reduction of the canonical
   system. Two different field theories landed in the same 1.69-neighborhood
   for H/Q on the electron point. Coincidences in 2-parameter calibration
   space are not extraordinary.

**Implication for Q36 (v1 citation in v9 paper §9):** the citation as
"independent reproduction" is even weaker than we thought yesterday.
v1 doesn't just use a different ansatz — it uses a different geometric
realization. Reply to Paul should ask him to remove or substantially
weaken the citation in any 9a revision.

| Particle | ω (script) | Predicted MeV | PDG MeV | Gap |

**Findings:**

+ DeepSeek's prescribed ω values (muon ω≈11, tau ω≈40.7) are *approximate*.
  Actual minima are ω=12.82 and ω=50.0.
+ **Muon at 0.80%** is a strong physics result — robust independent verification
  of the lepton spectrum scaling.
+ **Pion+ at 3.25%** is unexpected — pion is a quark-antiquark meson, not a
  lepton. Either the chaoiton ansatz also captures mesons (intriguing) or
  the match is coincidental (a meson mass happens to fall in the gap between
  muon and tau).
+ **Tau at 6.47%** is acceptable but weaker than muon. Suggests the H ∝ ω^β
  scaling isn't a perfect power law across the full range.

**Cylindrical r_rms saturates** — r_rms approaches a plateau ~1320 fm for
ω > 10, consistent with the cylindrical chaoiton having a characteristic
ring size that doesn't scale much with ω.
