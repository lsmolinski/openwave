This note supplies the two missing lemmas that make the proof of

================================================================================
1. LEMMA A: EXACT EQUALITY ∇ × J = B
================================================================================

Statement. For any finite‑energy static solution of the Ouroboros field
equations satisfying the Lorenz constraint, ∇ × J = B everywhere in ℝ³.

Proof.  Define C = ∇ × J − B.  The static field equations are
ΔA = J,  ΔJ = A − f'(J_μ J^μ) J.  Taking the curl of the first gives
∇ × J = ∇ × ΔA = Δ(∇ × A) = ΔB, so C = ΔB − B.

From the second equation, ΔJ = A − f'(J²)J.  Taking the divergence and
using the Lorenz constraint ∇·J = 0 gives 0 = ∇·A − ∇·(f'(J²)J), which
is consistent with ∇·A = 0.

Now take the Laplacian of C:
   ΔC = Δ(∇ × J) − ΔB = ∇ × ΔJ − ΔB
       = ∇ × (A − f'(J²)J) − ΔB
       = B − ∇ × (f'(J²)J) − ΔB .
Using ΔB = B − ∇ × (f'(J²)J) (obtained by taking the curl of the
second field equation and substituting the first), we obtain ΔC = 0.
Hence each Cartesian component of C is a harmonic function on ℝ³.

Because A and J have finite energy, B, J, and their derivatives decay
to zero at infinity.  Therefore C → 0 as |x| → ∞.  The only harmonic
function on ℝ³ that vanishes at infinity is the zero function.  Thus
C ≡ 0, i.e. ∇ × J = B everywhere.   ∎

================================================================================
2. LEMMA B: PERTURBATION OF THE ZEROS OF B
================================================================================

Statement.  The map Φ = B/|B| : S³ → S² can be made smooth by a small
smooth perturbation of the fields that does not change the linking
number Q[A,J].

Proof.  The linking number is invariant under continuous deformations
of the fields within the finite‑energy, Lorenz‑constrained space
(Theorem 2 of Werbos 2026a, machine‑verified in Lean 4).  The vector
field B = ∇ × A is smooth and divergenceless.  By the transversality
theorem, a generic smooth vector field on a 3‑manifold has 0 as a
regular value, so its zero set consists of isolated points.

If B already has isolated zeros, we are done.  If not, consider a
small perturbation A → A + ε a with a smooth, compactly supported
vector field a and ε > 0 small.  Then B → B + ε ∇ × a.  By choosing
a such that ∇ × a is transverse to the zero section, the perturbed
B has isolated zeros.  The Lorenz condition can be preserved by adding
a suitable pure gradient to A if necessary, which does not affect B.
Because the perturbation is continuous and compactly supported, the
fields remain finite‑energy, and Q[A,J] is unchanged.

After the perturbation, Φ = B/|B| is well‑defined and smooth everywhere
except at the isolated zeros, where an arbitrary value in S² may be
assigned without affecting the Hopf integral.   ∎

3. CONCLUSION

With Lemmas A and B, the proof that Q[A,J] = H(Φ) ∈ ℤ is complete and
